<img width="927" height="373" alt="QQ20260618-141314" src="https://github.com/user-attachments/assets/3789dd44-9e06-41c9-bfd8-5a462cc70fed" /># 👕 制衣厂销量数据看板 (Clothing Sales Dashboard)

这是一个基于 Streamlit 构建的交互式数据分析应用，旨在监控和分析制衣厂的每日生产与销售情况。

## 🚀 主要功能
- **多型号对比**：支持同时筛选多个产品型号进行趋势对比。
- **灵活日期查询**：自定义时间范围，精准定位特定周期的产量波动。
- **可视化图表**：使用 Plotly 绘制动态折线图，直观展示数据走势。
- **响应式设计**：完美适配 PC 端与移动端，随时随地查看数据。

## 🛠️ 技术栈
- **前端/交互**: Streamlit
- **数据处理**: Pandas
- **图表绘制**: Plotly Express
- **部署**: Streamlit Community Cloud
